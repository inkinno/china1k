# [개념 설계서 1] 듀오링고 스타일 XP 및 리그 제도 도입 & 네비게이션 확장 기획

## 1. 개요 및 기획 배경
현재 천자문 마스터는 1분당 1포인트를 부여하는 심플한 포인트 경제(Point Economy)를 채택하고 있습니다. 그러나 학습자의 장기 리텐션(재방문율 및 지속 학습 동기)을 극대화하기 위해서는 **듀오링고(Duolingo)**의 성공 요소인 **경쟁 기반 리그 제도(League System)**와 직관적인 **경험치(XP) 보상 체계**의 도입이 필수적입니다.

---

## 2. 포인트(P)와 경험치(XP)의 이원화 설계 (Economy Separation)
*   **포인트 (P - 재화)**: 상점(MVP)에서 소모성 아이템(출석 수호권, 펫 꾸미기, 오프라인 보상 교환)을 구매하는 통화(Currency).
*   **경험치 (XP - 명예)**: 주간 리그 승강격 및 유저 랭킹을 결정하는 순수 노력 지표(소모되지 않고 누적).

### [XP 획득 밸런싱 테이블]
| 학습 활동 | 지급 XP | 비고 |
| :--- | :--- | :--- |
| **플래시 카드 1자 열람 및 암기 완료** | +10 XP | 정독 학습 유도 |
| **일반 시험 1문제 정답** | +15 XP | 15문항 올클리어 시 총 225 XP |
| **오답노트 복습 1문제 정답** | +25 XP | 취약점 극복에 대한 가중치 부여 |
| **천자문 획따라 쓰기 1자 완료** | +20 XP | 쓰기 연습 참여 촉진 |
| **연속 정답 콤보 보상** | +5 ~ +50 XP | 5콤보(+10 XP), 10콤보(+30 XP), 15콤보(+50 XP) |
| **일일 첫 접속 (출석 보너스)** | +50 XP | 연속 출석(Streak) 일수 × 5 XP 추가 보너스 |

## 3. 주간 리그 티어 구조 및 승강격 규칙 (League Tiers)
듀오링고의 랭킹 시스템을 벤치마킹하여 총 **5단계 프리미엄 티어**로 운영합니다. 매주 월요일 오전 0시(KST)에 리그가 초기화되며 새로운 그룹이 결성됩니다.

```
[브론즈 리그] ──> [실버 리그] ──> [골드 리그] ──> [플래티넘 리그] ──> [천자문 마스터 리그]
  (입문자)         (일반 학습자)     (상위 30%)       (상위 10%)          (최상위 3% 명예)
```

1.  **리그 그룹 조기 매칭**: 매주 첫 학습을 시작한 유저 30명을 실시간으로 한 그룹(Room)으로 매칭합니다.
2.  **승급(Promotion)**: 매주 일요일 자정 기준, 그룹 내 **상위 5명(1~5위)**은 다음 상위 티어로 승급합니다.
3.  **유지(Retention)**: **6위 ~ 20위**는 현재 리그 티어를 유지합니다.
4.  **강등(Demotion)**: **21위 ~ 30위**(하위 10명)는 이전 하위 티어로 강등됩니다.
5.  **보상 지급**: 상위 1~3위 유저에게는 상점 재화인 **골드 포인트(P) 및 한정판 펫 칭호/배지**를 대량 지급합니다.

## 4. 네비게이션 및 UI/UX 구조 개편안
현재의 하단 5탭 구조를 확장하거나, 네비게이션 탭의 순서와 시각적 가중치를 조정하여 리그 참여도를 높입니다.

### [하단 네비게이션 바 제안]
1.  **[진도 학습]** (홈/메인) - 가장 자주 쓰는 플래시 카드 및 획쓰기 학습실
2.  **[시험/평가]** - 15문항/30문항 테스트 및 오답노트 연동
3.  **[리그 랭킹] (NEW)** - 실시간 주간 내 순위, 라이벌과의 XP 격차 그래프, 승강격 존 표시
4.  **[펫/보스전] (NEW)** - 학습으로 키운 펫 및 보스레이드 참여 화면
5.  **[상점/마이페이지]** - 아이템 구매 및 내 정보/배지/출석부 확인

## 5. 아키텍처 및 데이터베이스 구현 설계 (SOLID / DRY 원칙 준수)
*   **Firestore 데이터 구조**:
    *   `/artifacts/{appId}/leagues/{year_week}/groups/{groupId}`: 30명 단위의 리그 그룹 데이터
    *   `/artifacts/{appId}/users/{userId}/league/status`: 현재 속한 리그 티어, 주간 누적 XP, 순위 정보
*   **모듈화 설계**:
    *   `js/services/LeagueService.js`: 주간 XP 누적, 매칭 알고리즘, 순위 계산 전담 (Single Responsibility)
    *   `js/views/league.js`: 리그 랭킹 보드 렌더링 및 애니메이션 UI 컴포넌트
