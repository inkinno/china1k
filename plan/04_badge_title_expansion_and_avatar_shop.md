# [개념 설계서 4] 10배 확장 배지 & 칭호 장착 시스템 및 아바타 커스텀 상점 경제 기획

## 1. 개요 및 기획 배경
학습자의 장기적인 동기부여와 수집 욕구(Collection Desire)를 극대화하기 위해, 기존의 단순한 보상 체계를 **[경험치/재화 보완] -> [100여 개의 확장된 배지 및 칭호 획득] -> [아바타 커스텀 상점 소비]**로 이어지는 완벽한 보상 선순환 루프(Reward Loop)로 고도화합니다. 
본 기획서는 당장 구현하지 않고 차후 단계별(Phase)로 순차 적용할 수 있도록 독립된 아키텍처 및 경제 설계안을 제시합니다.

---

## 2. XP 및 재화(포인트) 획득 밸런싱 보완 & 퀘스트 루프
단순 반복 학습의 지루함을 없애고 다채로운 학습 목표를 제시하기 위해 **일일/주간 퀘스트 시스템**과 **활동별 가중치 리워드**를 도입합니다.

### [1] 일일 및 주간 퀘스트 (Quest System)
| 퀘스트 종류 | 목표 설명 | 보상 내용 | 비고 |
| :--- | :--- | :--- | :--- |
| **일일 퀘스트 1** | 플래시 카드 30자 이상 학습 | +50 P, +30 XP | 매일 자정 초기화 |
| **일일 퀘스트 2** | 일반 시험 또는 오답 퀴즈 1회 완수 | +80 P, +50 XP | 실전 점검 유도 |
| **일일 퀘스트 3** | 천자문 획따라 쓰기 5자 완수 | +70 P, +40 XP | 쓰기 감각 유지 |
| **주간 퀘스트** | 주간 5일 이상 출석 달성 | +300 P, +200 XP, [수호권 1개] | 주간 지속 학습 마일스톤 |

### [2] 동적 보너스 리워드 가중치
*   **퍼펙트 올클리어 보너스**: 시험에서 100점 달성 시 획득 포인트 1.5배 증폭.
*   **오답 완치 보너스**: 오답노트에 등록된 한자를 연속 3회 정답 맞추어 목록에서 제거할 때마다 +15 P 추가 지급.

## 3. 10배 확장 배지(Badge) & 칭호(Title) 장착 시스템
기존 소수의 배지를 **총 100개 이상의 정교한 업적 트리(Achievement Tree)**로 10배 이상 대폭 확장합니다. 획득한 배지 중 가장 자랑스러운 업적을 선택하면 프로필과 리그 랭킹 보드에서 닉네임 앞에 **[특별 칭호(Title)]**로 장착되어 과시할 수 있습니다.

### [1] 5대 업적 카테고리 (100+ 마일스톤 설계 예시)

#### ① 암기 및 진도 마일스톤 (Progress Mastery)
*   `first_step`: **[입문 학도]** - 천자문 첫 플래시 카드 학습 완료
*   `hanja_50`: **[오십보 백보]** - 50자 암기 완료
*   `hanja_300`: **[삼백자 지기]** - 300자 암기 완료
*   `hanja_500`: **[반환점 도달]** - 500자 암기 달성
*   `hanja_1000`: **[👑 천자문 대장인]** - 1,000자 완벽 마스터 (신화 등급)
*   `writing_100`: **[서예 장인]** - 획따라 쓰기 100자 완료

#### ② 성실 및 연속 출석 마일스톤 (Streak & Diligence)
*   `streak_3`: **[작심삼일 극복]** - 3일 연속 출석
*   `streak_7`: **[일주일의 기적]** - 7일 연속 출석
*   `streak_30`: **[🔥 불타는 열정]** - 30일 연속 출석
*   `streak_100`: **[백일의 기도]** - 100일 연속 출석
*   `streak_365`: **[🌟 전설의 학우]** - 365일(1년) 연속 출석 (전설 등급)
*   `night_owl`: **[새벽을 여는 자]** - 밤 12시~새벽 4시 사이 학습 10회 달성

#### ③ 시험 및 정확도 마일스톤 (Accuracy & Quiz)
*   `quiz_perfect_1`: **[백발백중]** - 일반 시험 100점 첫 달성
*   `quiz_perfect_10`: **[신출귀몰 시험관]** - 100점 10회 달성
*   `combo_30`: **[폭풍의 콤보]** - 퀴즈 30연속 정답
*   `wrong_remedy_50`: **[오답 부활자]** - 오답노트에서 50개 한자 완치(탈출) 성공

#### ④ 리그 및 경쟁 마일스톤 (League & Social)
*   `league_gold`: **[황금 리거]** - 골드 리그 승급
*   `league_master`: **[🏆 리그 지배자]** - 최상위 마스터 리그 도달
*   `league_winner`: **[주간 챔피언]** - 주간 리그 그룹 최종 1위 달성
*   `social_butterfly`: **[인맥 왕]** - 친구 10명 이상 등록 및 동반 매칭 달성

#### ⑤ 이색 및 히든 업적 (Hidden & Fun)
*   `speed_racer`: **[광속 학도]** - 1분에 플래시 카드 30자 이상 낭독 및 열람
*   `shop_first`: **[첫 소비의 맛]** - 상점에서 첫 아이템 구매
*   `seven_fall`: **[칠전팔기]** - 퀴즈에서 7번 틀린 한자를 마침내 완치했을 때 획득

## 4. 아바타 커스텀 상점 (Avatar Customization Shop)
학습을 통해 땀 흘려 모은 포인트(P)를 가장 가치 있고 재미있게 소비할 수 있는 **[아바타 꾸미기 시스템]**을 상점의 핵심 콘텐츠로 설계합니다.

### [1] 아바타 4대 꾸미기 슬롯 (Customization Slots)
유저의 기본 프로필 아바타 주변을 4가지 파츠로 다채롭게 조합하여 나만의 개성을 표현합니다.

```
       [2. 헤어/모자 (Headgear)]  --> 예: 유생 관모, 황제의 면류관, 귀여운 학사모
       [1. 테두리 프레임 (Frame)] --> 예: 청동 거울, 황금 옥새, 네온 홀로그램
         ( 아바타 얼굴 )        --> 기본 다이스베어 아바타 표정/스타일
       [3. 배경 아우라 (Aura)]    --> 예: 수묵화 효과, 불타는 화염, 흩날리는 벚꽃
       [4. 네임플레이트 명찰]     --> 예: 그라데이션 닉네임 색상, 반짝이는 명찰
```

### [2] 상점 아이템 라인업 및 희귀도 경제 (Rarity Tier)
아이템은 희귀도에 따라 가격 및 시각적 이펙트(동적 CSS 애니메이션, 파티클 등)가 차별화됩니다.

| 희귀도 등급 | 아이템 예시 | 구매 가격 | 시각 이펙트 및 특징 |
| :--- | :--- | :--- | :--- |
| **일반 (Common)** | 나무 원목 프레임, 짚신, 붓단청 명찰 | 300 P | 단정하고 고전적인 고전 서당 스타일 |
| **희귀 (Rare)** | 유생 관모, 청동 거울 테두리, 수묵화 아우라 | 1,000 P | 은은하게 빛나는 테두리 하이라이트 |
| **영웅 (Epic)** | 장원급제 어사화, 황금 옥새 프레임, 대나무 파티클 | 3,500 P | 마우스 호버 시 벚꽃/대나무 잎이 흩날리는 애니메이션 |
| **전설 (Legendary)** | 황제의 면류관, 화염룡의 아우라, 사이버 유생 고글 | 10,000 P | 상시 불타오르거나 오로라 렌즈 빛 반사 이펙트 |
| **신화 (Mythic)** | **[천공의 마스터 옥좌]** (리그 1위 한정판) | 리그 상점 전용 | 프로필 테두리에 황금 용이 감싸며 움직이는 최강 칭호 |

## 5. 데이터 아키텍처 및 확장 설계 (SOLID / DRY 원칙)
추후 본 기획을 코드로 구현할 때 구조적 단순함과 확장성을 유지하기 위한 상태 메타디자인입니다.

*   **`state.js` 스키마 확장안**:
    ```javascript
    userProfile: {
      equippedTitle: "hanja_1000",       // 현재 장착 중인 칭호 배지 ID
      avatarCustom: {
        frame: "frame_gold_seal",        // 장착된 프레임 ID
        headgear: "head_scholar_hat",    // 장착된 모자 ID
        aura: "aura_blossom",            // 장착된 배경 효과 ID
        nameplate: "name_neon_blue"      // 장착된 명찰 ID
      },
      inventory: [                       // 구매/획득한 꾸미기 아이템 보관함
        "frame_wood", "frame_gold_seal", "head_scholar_hat", "aura_blossom"
      ]
    }
    ```
*   **모듈 역할 분담**:
    *   `js/data/badges.js`: 100여 개 배지의 메타데이터(ID, 이름, 설명, 아이콘, 달성 조건 함수)를 선언적 배열로 관리 (DRY 원칙).
    *   `js/data/shopItems.js`: 아바타 꾸미기 아이템의 가격, 희귀도, CSS 스타일 속성을 정의.
    *   `js/views/shop.js`: 미리보기(Preview) 아바타 캔버스 및 부위별 탭 쇼핑몰 UI 렌더링.
